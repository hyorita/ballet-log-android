# Ballet Log 1.6 — 사이드로드 사용자 안내

원스토어 사용자에게 직접 APK를 전달할 때 함께 보낼 안내 문구.
APK 파일: `app/build/outputs/apk/release/BalletLog-1.6.apk`

---

## 사용자에게 보낼 메시지 (복붙용)

🩰 Ballet Log 1.6 업데이트 안내

새 기능 (포토 로그, 백업 등)을 미리 받을 수 있는 APK 보내드립니다.
원스토어 버전과 동일한 서명이라 업데이트 형태로 설치되고
기존 데이터는 그대로 유지됩니다.

1️⃣ 설치하기
- APK 파일 탭
- "출처를 알 수 없는 앱 설치 허용" 요청 시 허용
- 설치 후 자동으로 업데이트됨 (기존 데이터 그대로)

2️⃣ 새 기능 둘러보기
- 포토 로그 탭 (왼쪽 첫 번째)
- 메모 탭 (사진 첨부, 클래스 연결 등)
- 톱니바퀴 → 설정 (프로필, 백업)

3️⃣ ⚠️ 백업 — 가장 중요!
나중에 플레이스토어 정식 버전으로 옮길 때를 대비해
지금 데이터를 백업해두세요:

  포토 로그 → 톱니바퀴 → 백업 → 데이터 내보내기

저장 위치는 반드시 외부에 (구글 드라이브, 이메일, 카카오톡 자기에게 보내기 등).
앱 안에 저장하면 나중에 앱 삭제될 때 같이 사라집니다.

4️⃣ (나중에) 플레이스토어 버전으로 이동
플레이스토어에 정식 출시되면:
- 이 APK로 설치한 앱은 한 번 삭제 (서명 다름)
- 플레이스토어에서 새로 설치
- 설정 → 백업 → 데이터 가져오기 → 위에 저장한 백업 파일 선택
- 자동 재시작 후 데이터 복원

문제 있으면 알려주세요.
